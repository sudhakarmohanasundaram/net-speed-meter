# Network Speed Meter

A lightweight Windows network speed meter that displays live download and upload speed in Mbps near the taskbar.

## Features
- Real-time download speed
- Real-time upload speed
- Compact floating meter
- Draggable window
- Lightweight and fast

## Run
python taskbar_meter.py

## Build EXE
pyinstaller --onefile --noconsole taskbar_meter.py
